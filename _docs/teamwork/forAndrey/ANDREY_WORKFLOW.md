## 🎯 Твой процесс работы

### **Шаг 1: Работаешь в Windsurf**

1. Открываешь проект в Windsurf
2. Делаешь изменения в UI:
   - `frontend/src/components/` - компоненты
   - `frontend/src/pages/` - страницы
   - `frontend/src/styles/` - стили
   - Промпты для AI (если нужно)

### **Шаг 2: Коммитишь изменения**

```bash
git add .
git commit -m "feat(ui): добавил новый компонент"
```

**Формат сообщений:**
- `feat(ui):` - новый UI компонент
- `feat(prompts):` - изменения в промптах
- `fix(ui):` - исправление бага
- `style(ui):` - стили, CSS

### **Шаг 3: Пушишь в GitHub**

```bash
git push origin main
```

### **Шаг 4: Деплоишь на test**

В Windsurf набираешь команду:

```
/andrey-test
```

Эта команда:
1. Коммитит твои изменения
2. Пушит в GitHub
3. Подключается к серверу
4. Скачивает изменения
5. Собирает frontend
6. Перезапускает backend (если нужно)
7. Готово!

### **Шаг 5: Тестируешь на test.optioner.online**

Открываешь: https://test.optioner.online

Проверяешь:
- ✅ UI отображается правильно
- ✅ Все кнопки работают
- ✅ Нет ошибок в консоли браузера
- ✅ Промпты работают (если менял)

### **Шаг 6: Если все ОК**

Пишешь Левону: "Готово, можно мержить в production"

Левон проверит и задеплоит на production (optioner.online)

## ⚠️ Важные правила

### **✅ Можно:**
- Деплоить на test сколько угодно раз
- Экспериментировать на test
- Ломать test (это для тестов!)
- Работать в main ветке (пока проект небольшой)

### **❌ Нельзя:**
- Деплоить на production (это делает Левон)
- Менять backend код (это зона Левона)
- Менять конфигурацию сервера
- Удалять файлы без согласования

### **🤝 Взаимодействие с Левоном:**
- Если нужен новый API endpoint → пиши Левону
- Если баг в backend → пиши Левону
- Если нужно на production → пиши Левону
- Если что-то непонятно → пиши Левону

## 🐛 Что делать если что-то пошло не так

### **Проблема: Деплой не работает**

```bash
# Проверь, что запушил изменения
git status
git push origin main

# Попробуй еще раз
/andrey-test
```

### **Проблема: Сайт не открывается**

1. Подожди 1-2 минуты (сборка frontend занимает время)
2. Обнови страницу (Ctrl+F5 или Cmd+Shift+R)
3. Проверь консоль браузера (F12)
4. Напиши Левону

### **Проблема: Ошибка в консоли**

1. Открой консоль браузера (F12)
2. Сделай скриншот ошибки
3. Отправь Левону

### **Проблема: Backend не отвечает**

1. Это зона Левона
2. Напиши Левону с описанием проблемы
