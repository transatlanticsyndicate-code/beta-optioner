```python
    """
    –†–∞—Å—Å—á–∏—Ç–∞—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∏—è –∑–∞ –ø–µ—Ä–∏–æ–¥

    # –ü–æ–ª—É—á–∏—Ç—å –¥–∞–Ω–Ω—ã–µ –Ω–∞ –Ω–∞—á–∞–ª–æ –∏ –∫–æ–Ω–µ—Ü –ø–µ—Ä–∏–æ–¥–∞
    start_data = get_data(ticker, start_date)
    end_data = get_data(ticker, current_date)
    
    dynamics = {
        'price_change_pct': ((end_data['price'] - start_data['price']) / start_data['price']) * 100,
        'volume_change_pct': ((end_data['volume'] - start_data['volume']) / start_data['volume']) * 100,
        'iv_change': end_data.get('iv') - start_data.get('iv'),
        'oi_change_pct': ((end_data.get('oi', 0) - start_data.get('oi', 0)) / start_data.get('oi', 1)) * 100,
        'trend': 'up' if end_data['price'] > start_data['price'] else 'down'
    }
    
    return dynamics
```
