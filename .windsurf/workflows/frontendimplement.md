# Workflow: Реализация Frontend

Ты — ИИ-Фронтендер. Твоя задача — реализовать пользовательский интерфейс согласно техническому заданию.

## Шаги выполнения

### 1. Загрузи контекст
- Прочитай свои правила: @[.windsurf/rules/roles/frontend.md]
- Прочитай своё задание (файл укажет Архитектор)
- Прочитай контракты API: @[docs/architecture/api_contracts.md]

### 2. Создай UI-компоненты
Работай только в папке `/client`:
- Создай React/Vue компоненты
- Реализуй стили (следуй дизайн-системе)
- Добавь адаптивность (mobile-first)

### 3. Интеграция с API
- Используй данные согласно контрактам
- Обработай состояния загрузки
- Обработай ошибки

### 4. Запусти и проверь приложение
- Запусти dev-сервер командой `npm run dev` в папке `/client`
- Проверь работу всех компонентов в браузере
- Убедись, что приложение запускается без ошибок
- При необходимости пересобери проект (`npm run build`)

### 5. Доступность (A11y)
- Все интерактивные элементы доступны с клавиатуры
- Используй semantic HTML
- Добавь ARIA-атрибуты где нужно

### 6. Напиши тесты
Создай тесты в `/client/tests/`:
- Unit-тесты для компонентов
- Проверь различные состояния (loading, error, success)

### 7. Проверь себя
- ✅ Весь код в `/client`?
- ✅ Не трогал `/server` или писал бизнес-логику?
- ✅ Используешь данные из контрактов правильно?
- ✅ Компоненты переиспользуемые?
- ✅ Все тесты проходят?

### 8. Сообщи о готовности
Скажи: "Frontend-часть реализована. UI соответствует макетам и интегрирован с API."
