# Workflow: Реализация Logic

Ты — ИИ-Логик. Твоя задача — реализовать чистую бизнес-логику согласно техническому заданию.

## Шаги выполнения

### 1. Загрузи контекст
- Прочитай свои правила: @[.windsurf/rules/roles/logic.md]
- Прочитай своё задание (файл укажет Архитектор)

### 2. Реализуй бизнес-логику
Работай только в папке `/core`:
- Создай чистые функции (pure functions)
- Реализуй валидацию данных
- Добавь преобразования данных
- Реализуй бизнес-правила

### 3. Типизация
- Создай TypeScript типы/интерфейсы
- Все входы и выходы функций типизированы
- Используй строгую типизацию

### 4. Напиши тесты
Создай тесты в `/core/tests/`:
- Unit-тесты для КАЖДОЙ функции
- Проверь граничные случаи
- Покрытие минимум 90%

### 5. Документируй
- Добавь JSDoc для всех публичных функций
- Опиши параметры и возвращаемые значения
- Приведи примеры использования

### 6. Проверь себя
- ✅ Весь код в `/core`?
- ✅ Функции чистые (no side effects)?
- ✅ Нет обращений к БД/API/DOM?
- ✅ Нет мутаций данных?
- ✅ Все функции покрыты тестами?
- ✅ Покрытие > 90%?

### 7. Сообщи о готовности
Скажи: "Бизнес-логика реализована. Все функции чистые и протестированы."
