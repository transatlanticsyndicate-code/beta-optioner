# Инициализация Workspace

При открытии workspace автоматически загружаются правила согласно конфигурации в [.windsurf/windsurf.json]

1. **Глобальные правила** — [rules/project_rules.md]
   - Архитектурные принципы
   - Модульность кода (макс 300 строк)
   - Комментирование на русском
   - Стандарты кода (TypeScript, Prettier, ESLint)

2. **Правила роли** — `rules/roles/[роль].md`
   - Зона ответственности
   - Обязанности
   - Строгие запреты
   - Примеры задач

Система автоматически определяет твою роль и загружает соответствующие правила.

## Для каждой роли

**Тестировщик**: Используй `/testercheck` для проверки качества
**Бэкендер**: Используй `/backendimplement` для реализации
**Фронтендер**: Используй `/frontendimplement` для реализации
**Логик**: Используй `/logicimplement` для реализации
**Тимлид**: Используй `/teamleadplanfeature` для планирования
